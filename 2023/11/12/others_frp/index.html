二、准备frp

进入github，下载适合的frp:https://github.com/fatedier/frp/releases (如：https://github.com/fatedier/frp/releases/download/v0.15.1/frp_0.15.1_linux_amd64.tar.gz)

2.1 本机

cd /home/zhifeng/software/frp_0.15.1_linux_amd64

2.2 远程服务器

登录到腾讯云服务器，解压。

frps 是服务器对应的执行文件，frps.ini 是服务器配置文件。

修改 frps 配置文件： vim frps.ini

这个 bind_port 就是 frps 的监听端口，可以修改为你需要制定的端口，在云服务器中，我们一般不会使用默认端口，因为这可能会导致出现病毒攻击。

腾讯云设置：找到防火墙管理规则，开放暴露端口。

注意

如果你要通过 frp 进行 ssh 访问，还要开启一个 ssh 访问端口，也就是要暴露两个端口，一个 7001 端口，一个 7001 端口，其中 8001 端口用于提供 ssh 服务。（也就是防火墙开两个端口，一个访问一个监听）

总结，修改配置文件，开放两个端口。

如果执行报错，可以执行 chmod 777 frps 增加权限。

五、设置开机自启动

5.1、云服务器

方法一：设置后台挂载
nohup ./frps -c frps.ini &

方法二：云服务器：
systemctl start frps
systemctl enable frps
查看日志
systemctl status frps

5.2、远程的机子（本机）

方法一：

sudo vim /lib/systemd/system/frps.service

输入下面内容：
[Unit]
Description=Frp Client Service
After=network.target

[Service]
Type=simple
User=zhifeng
Restart=on-failure
RestartSec=5s
ExecStart=/home/zhifeng/software/frp_0.15.1_linux_amd64/frpc -c /home/zhifeng/software/frp_0.15.1_linux_amd64/frpc.ini
ExecReload=/home/zhifeng/software/frp_0.15.1_linux_amd64/frpc reload -c /home/zhifeng/software/frp_0.15.1_linux_amd64/frpc.ini
LimitNOFILE=1048576

[Install]
WantedBy=multi-user.target

然后开启：（有时候需要重载systemctl daemon-reload）
sudo systemctl start frps
sudo systemctl enable frps
sudo systemctl restart frps
sudo systemctl status frps
