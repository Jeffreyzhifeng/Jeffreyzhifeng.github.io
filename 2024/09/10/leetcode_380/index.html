leetcode_380.O(1)时间插入、删除和获取随机元素

Leetcode
Created At : 2024-09-10 12:16
Views 👀 :
Comment:

实现RandomizedSet 类：

  • RandomizedSet() 初始化 RandomizedSet 对象
  • bool insert(int val) 当元素 val 不存在时，向集合中插入该项，并返回 true ；否则，返回 false 。
  • bool remove(int val) 当元素 val 存在时，从集合中移除该项，并返回 true ；否则，返回 false 。
  • int getRandom() 随机返回现有集合中的一项（测试用例保证调用此方法时集合中至少存在一个元素）。每个元素应该有 相同的概率 被返回。

你必须实现类的所有函数，并满足每个函数的 平均 时间复杂度为 O(1) 。

示例：

题解：

为了满足 O(1) 时间复杂度的要求，使用以下两种数据结构的组合是关键：

  1. **哈希表 (Dictionary/Map)**：哈希表能够在 O(1) 时间内检查元素是否存在、插入元素和删除元素。我们用它来存储值及其对应的索引。
  2. **动态数组 (List)**：数组可以在 O(1) 时间内获取随机元素（通过随机选择数组中的索引）。

哈希表和数组的组合能够解决两个关键问题：

  • 插入和删除操作的 O(1) 时间复杂度：通过哈希表可以快速查找和操作元素。
  • 随机访问的 O(1) 时间复杂度：数组通过索引直接访问随机元素。

插入时：

  • 检查元素是否在哈希表中，如果不存在，加入数组并在哈希表中记录它的索引。

删除时：

  • 为了保证 O(1) 时间复杂度，我们可以先将要删除的元素与数组中的最后一个元素交换，然后删除数组的最后一个元素，这样避免了移除数组中间元素带来的 O(n) 开销。

随机返回时：

  • 随机从数组中返回一个元素，因为数组的每个元素都有相同的被选中概率。
